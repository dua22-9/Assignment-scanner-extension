# Assignment-scanner-extension
📌 Overview
Managing assignments across multiple subjects can be frustrating for students. Often, we forget deadlines mentioned in class or get lazy about clicking through each subject just to check pending uploads.
This browser extension solves that problem by automatically scanning the student portal and showing assignments with deadlines right away. Whenever you open your portal, it acts as a reminder of what’s due in the submission section.
It may not save a huge amount of time, but it definitely saves effort — and effort matters when you’re juggling classes, deadlines, and student life.
✨ Features
- Automatic Assignment Detection: Reads the portal and finds assignments in the upload section.
- Deadline Remin- der: Displays due dates instantly when you open the portal.
- Effort Saver: No need to click through each subject individually.
🚀 Future Plans
This is just one part of a larger project. More updates will follow, including features to make academic workflow even smoother.
🛠️ Tech Stack
- Browser Extension APIs
- JavaScript / HTML / CSS
- DOM manipulation for portal data extraction
🎯 Why This Matters
As students, we often struggle to manage everything at once. This extension acts as a small but meaningful helper, keeping assignments visible and reducing the chance of missing deadlines.
